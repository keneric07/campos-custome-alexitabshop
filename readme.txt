Plugin privado para WooCommerce que permite pedir nombre, número y foto por cada unidad comprada.

== Funciones ==
- Activa la personalización por producto desde el editor de WooCommerce.
- Genera campos dinámicos según la cantidad elegida.
- Solicita nombre, número y foto por cada unidad.
- Valida JPG, PNG o WebP.
- Límite máximo por foto: 2.5 MB.
- Guarda los datos en carrito, checkout y pedido.
- Bloquea el cambio de cantidad en carrito para evitar personalizaciones incompletas.
- Cambia el botón del catálogo por "Personalizar" para llevar al cliente a la página del producto.

== Instalación ==
1. En WordPress entra a Plugins > Añadir nuevo > Subir plugin.
2. Sube el ZIP alexita-product-personalizer.zip.
3. Activa el plugin.
4. Edita el producto que quieres personalizar.
5. En Datos del producto > General, marca "Personalización Alexita".
6. Guarda o actualiza el producto.

== Recomendación ==
Prueba primero en un producto de prueba antes de activarlo en productos reales.
